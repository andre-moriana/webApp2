LOGO FFTA - Documents d'édition concours
========================================

Pour afficher le logo officiel FFTA dans les documents d'édition, placez le fichier
du logo dans ce dossier sous le nom : ffta-logo.png

CHARTE GRAPHIQUE FFTA
---------------------
Le logotype FFTA est déposé à l'INPI. Son utilisation est régie par la charte graphique :
https://www.ffta.fr/sites/default/files/2024-11/charte_graphique_ffta.pdf

Points clés :
- Taille minimale : 15 mm
- Espace libre autour du logo : au minimum la hauteur du lettrage FFTA
- Couleurs officielles : Bleu Pantone 287, Rouge Pantone 032, Noir
- Aucune utilisation sans accord préalable des responsables FFTA

OBTENIR LE LOGO
---------------
Pour toute utilisation du logo, contacter le service Communication de la FFTA :
- Site : www.ffta.fr
- E-mail : ffta@ffta.fr
- Adresse : 268/270 rue de Brément - 93561 Rosny sous Bois Cedex

FORMAT RECOMMANDÉ
-----------------
PNG ou JPG, fond blanc. Préférer PNG pour une meilleure qualité d'impression.
Le fichier doit être nommé : ffta-logo.png
